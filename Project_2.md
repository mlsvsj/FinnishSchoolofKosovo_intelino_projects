## 🚂 Project 2: Train Station  

### 🧠 About Scratch  
Scratch is a visual programming tool where we use blocks instead of typing code.  
Each block gives instructions to control the train step by step.


### 📘 Project Description  
In this project, the train behaves like a real train station.  
It moves forward, stops at a specific distance, and then continues moving.

### 🧱 Code  
when movement forward  
reset distance  
drive forward at 30 cm/s  
wait until distance > 50  
stop driving  
pause driving for 3 seconds  
drive forward at 30 cm/s  

### 🚂 What Happens  
- The train starts moving  
- It measures the distance  
- It stops like at a station  
- After a short pause, it moves again  

### 🎯 Student Task  
Make the train move and stop like a real station.  
Try changing the distance where the train stops.  
You can also change how long the train waits before moving again.

### ❓ Questions  
- Why do we use “reset distance”?  
- What happens if we change the distance value?  
- How can we make the stop longer?
