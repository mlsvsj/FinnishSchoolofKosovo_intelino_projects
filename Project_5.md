## 🚂 Project 5: Smart Train System  

### 🧠 About Scratch  
Scratch allows us to combine movement, distance, and LED lights to create smart behaviors.

---

### 📘 Project Description  
In this project, the train becomes “smart” by combining movement, stops, and LED signals.

---

### 🧱 Code  
```
when movement forward  
reset distance  
drive forward at 30 cm/s  
set top LED color to green  
wait until distance > 50  
stop driving  
set top LED color to red  
pause driving for 3 seconds  
drive forward at 30 cm/s  
set top LED color to blue  
```

---

### 🚂 What Happens  
- The train moves with green light  
- It stops and turns red  
- It waits like a station  
- It continues moving with a different color  

---

### 🎯 Student Task  
Create a smart train using movement and LED lights.  
Try using different colors for different actions.  
You can add more stops and more LED changes.

---

### ❓ Questions  
- What does each LED color represent?  
- Why does the train stop?  
- How can we make it more “smart”?
