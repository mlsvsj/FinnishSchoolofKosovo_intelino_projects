# 🚂 Project 1: Move → Wait → Move

## About Scratch
Scratch is a visual programming tool where we use blocks instead of typing code.  
Each block is an instruction that tells the train what to do step by step.

## Project Description
In this project, we control the intelino train using simple blocks.  
The train starts moving, then stops, and then moves again.

## Code
when movement forward  
drive forward at 30 cm/s  
pause driving for 5 seconds  
drive forward at 30 cm/s  

## 🎯 Student Task
Create the same project and change speed or time.
