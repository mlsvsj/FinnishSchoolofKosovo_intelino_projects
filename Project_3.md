## 🚂 Project 3: LED Light Train  

### 🧠 About Scratch  
Scratch allows us to control not only movement but also lights and actions using blocks.

---

### 📘 Project Description  
In this project, the train uses LED lights to show what it is doing.  
Different colors represent different actions.

---

### 🧱 Code  
```
when movement forward  
drive forward at 30 cm/s  
set top LED color to green  
pause driving for 3 seconds  
set top LED color to red  
stop driving  
```

---

### 🚂 What Happens  
- The train starts moving  
- The LED turns green while moving  
- The train stops  
- The LED changes to red  

---

### 🎯 Student Task  
Make the train move and use different LED colors.  
Try changing the colors for different actions.  
You can also add more color changes while the train moves.

---

### ❓ Questions  
- What does the green light mean?  
- What happens when the light turns red?  
- Can you add another color?
