# 🚂 Intelino Train Projects

## 🏫 Finnish School of Kosovo  
**Teacher:** Melisa Vasija  

## 📘 Description
This repository presents ICT projects developed at the Finnish School of Kosovo using the intelino smart train.  
Students learn coding through hands-on activities by controlling movement, distance, speed, and LED lights.  
The goal is to introduce programming concepts in a fun, interactive, and real-world learning environment.

## 📂 Projects

### 🚂 Project 1: Move → Wait → Move  
Basic train movement with pause.

### 🚂 Project 2: Train Station  
Train stops at a specific distance like a real station.

### 🚂 Project 3: LED Light Train  
Train uses LED colors to show movement and actions.

### 🚂 Project 4: Two Stations  
Train stops at two different points on the track.

### 🚂 Project 5: Smart Train System  
Combination of movement, distance, and LED behavior.
